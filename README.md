این اسکریپت ماشین حساب هست پیشرفته ولی باگ هایی دارد در حل مسعله یکم مشکل هست اما کاربردیست

با دستور git clone https://github.com/sinashabani08-maker/MASHIN.git

با دستور cd MASHIN وارد پوشه شوید

با دستور python3 mashin.py اسکریپت را اجرا کنید 

لطفا به هیچ کدام از اسکریپت ها کاری نگیرید

و هرگونه کپی برداری و انتشار به نام خود پیگرد قانونی دارد
